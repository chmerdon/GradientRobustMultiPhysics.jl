Function Operators

FunctionOperators are building blocks for the weak form and define the operations that should be applied to the trial and test functions (and their discrete representatives) inside some PDEOperator. Below is a list of currently available FunctionOperators. Note, that not all operators an be applied to all finite element types.

Function operatorDescription
Identityidentity
IdentityDisc{Jump}jumps of identity (only over faces)
IdentityDisc{Average}average of identity (only over faces)
IdentityComponent{c}identity of c-th component
NormalFluxnormal flux (function times normal)
TangentFluxtangent flux (function times tangent)
Gradientgradient/Jacobian (as a vector)
GradientDisc{Jump}jumps of gradient/Jacobian (only over faces)
GradientDisc{Average}average of gradient/Jacobian (only over faces)
SymmetricGradientsymmetric part of the gradient
Divergencedivergence
CurlScalarcurl operator 1D to 2D (rotated gradient)
Curl2Dcurl operator 2D to 1D
Curl3Dcurl operator 3D to 3D
HessianHesse matrix = all 2nd order derivatives (as a vector)
LaplacianLaplace Operator
ReconstructionIdentity{FEType}reconstruction operator into specified FEType
ReconstructionIdentityDisc{FEType,Jump}jump of reconstruction operator (over faces)
ReconstructionIdentityDisc{FEType,Average}average of reconstruction operator (over faces)
ReconstructionDivergence{FEType}divergence of FEType reconstruction operator
ReconstructionGradient{FEType}gradient of FEType reconstruction operator
ReconstructionGradientDisc{FEType,Jump}jump of reconstruction operator gradient (over faces)
ReconstructionGradientDisc{FEType,Average}average of reconstruction operator gadient (over faces)
Note

Especially note the operators Reconstruction...{FEType} operators that allow to evaluate operators of some reconstructed version of a vector-valued testfunction that maps its discrete divergence to the divergence and so allows e.g. gradient-robust discretisations with classical non divergence-conforming ansatz spaces. So far such operators are available for the vector-valued Crouzeix-Raviart and Bernardi–Raugel finite element types.

Note

As each finite element type is transformed differently from the reference domain to the general domain, the evaluation of each function operator has to be implemented for each finite element class. Currently, not every function operator works in any dimension and for any finite element. More evaluations are added as soon as they are needed (and possibly upon request).
